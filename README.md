GetMACAddress
=============

得到本机的MAC地址

计算机网络实验:MAC地址的读取
-------------

查阅有关LAN的文档资料，特别是以太网（IEEE802.3 CSMA/CD MAC）标准，采用所掌握的程序设计语言，编写代码，输出本机以太网适配器（网络接口卡）的MAC地址，以16进制形式表示。
